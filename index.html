About this tool

Images are compressed in-browser before upload — no quality loss visible to viewers.
Files are stored in Supabase Storage (S3-compatible), served via CDN.
You can upload multiple images at once. Each gets its own numbered title.
Use arrow keys in the gallery lightbox to navigate between photos.

Limits

Max file size: 10 MB per image
Accepted: JPG, PNG, WebP, GIF
Images resized to max 2000px before upload
Free Supabase tier: 1GB storage, 2GB bandwidth/month
